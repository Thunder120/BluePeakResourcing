FAQ

The skeptical questions, answered.

How can your prices be this much lower than retail?
We buy through authorized distributors at wholesale margins and run a lean operation — no mall rent, no seasonal window displays. The savings pass through. We don't discount by cheapening the product.
Are these genuine products?
Yes. Every bottle is sourced through documented authorized channels. Each carries a batch code you can verify above. If a bottle ever fails an independent authentication test, we refund and pay for the test.
What's your return policy?
Unopened bottles: 14-day full refund. Opened bottles: only if authenticity is disputed and verified as such — fragrance is a hygiene product, so opened returns for preference reasons aren't accepted. [CONTENT: confirm actual policy]
How do I verify authenticity myself?
Use the batch lookup above, or send us the code before purchase and we'll pre-verify. Third-party authentication is also welcomed — we'll cover the cost if any bottle we sold fails.
Shipping timelines?
Metro deliveries: 2–4 business days via Shiprocket-routed carriers. Remote pincodes: 5–8 days. Track in real time on the order page. [CONTENT: confirm]
